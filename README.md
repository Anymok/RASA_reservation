# Requirements
 - Visual Studio Code or an IDE
 - Python 3.10



# Setup
 - Open project with your IDE
 - Open terminal and execute :
    - py -3.10 -m venv .env
    - .\.env\Scripts\activate
    - pip3 install rasa
    - py -m rasa train



# Start tchating with bot
 - Open project with your IDE
 - Open terminal and execute :
    - .\.env\Scripts\activate
    - py -m rasa shell
